This guide walks you through cloning the project, installing dependencies, and running the development server.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** v18 or later — [nodejs.org](https://nodejs.org)
- A package manager: **npm** (bundled with Node.js), **pnpm**, or **yarn**

<Tip>
  Run `node -v` and `npm -v` to verify your installed versions before proceeding.
</Tip>

## Setup

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/yourusername/allocazione.dev.git
    cd allocazione.dev
    ```
  </Step>

  <Step title="Install dependencies">
    Choose your preferred package manager:

    <CodeGroup>

    ```bash npm
    npm install
    ```

    ```bash pnpm
    pnpm install
    ```

    ```bash yarn
    yarn install
    ```

    </CodeGroup>
  </Step>

  <Step title="Start the development server">
    ```bash
    npm run dev
    ```

    Vite will start a local server, typically at `http://localhost:5173`.
  </Step>

  <Step title="Open in your browser">
    Navigate to `http://localhost:5173` to see the site running locally.

    <Note>
      Hot module replacement (HMR) is enabled by default. Changes to `.svelte`, `.ts`, and `.css` files will update the browser instantly without a full reload.
    </Note>
  </Step>
</Steps>

## Production build

The project uses `@sveltejs/adapter-static` to generate a fully static site.

<Steps>
  <Step title="Build for production">
    ```bash
    npm run build
    ```

    The output is written to the `build/` directory as static HTML, CSS, and JavaScript files ready for deployment.
  </Step>

  <Step title="Preview the production build">
    ```bash
    npm run preview
    ```

    This serves the contents of `build/` locally so you can verify the production output before deploying.

    <Warning>
      The preview server is not intended for production use. Use a static hosting provider such as Cloudflare Pages, Netlify, or Vercel to deploy the `build/` output.
    </Warning>
  </Step>
</Steps>

## Available scripts

| Script | Command | Description |
|--------|---------|-------------|
| Dev server | `npm run dev` | Starts Vite dev server with HMR |
| Build | `npm run build` | Compiles static output to `build/` |
| Preview | `npm run preview` | Serves `build/` locally for verification |
| Sync | `npm run prepare` | Runs `svelte-kit sync` to generate type definitions |
